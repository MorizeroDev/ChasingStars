# 目录结构

```
code\Morizero：代码主目录
   -Assets：
      --Drama：剧本对话框控制器
         ---Scripts：剧本脚本
      --Experiments：放置一些测试新功能用的东西
      --Fonts：游戏的字体
      --Icon：游戏的图标
      --Map：游戏地图
      --OBE：开箱体验
      --RainMaker：第三方雨天效果组件
      --Resources：
         ---BGM：背景音乐目录，地图控制器会从这里载入
         ---Characters：角色立绘
         ---Dramas：放置剧情预制体
         ---Map：放置地图块
            ----Prefabs：一些已经制作好的地图物件
         ---Players：人物行走图
         ---Plot：用于特殊背景/动画展示的情节预制体
         ---Prefabs：预制体
         ---Snd：音效
      --Shuttle：走廊音乐游戏（需要大改）
      --Startup：启动页面
      --UI：一些和全局UI相关的东西
release：编译成品（已在.gitignore中设置忽略）
```
