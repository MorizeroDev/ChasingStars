# 森之零点

404于2019.7完工的**森之物语**的续作，但剧情上无太多联系性

预测完工时间：**2022.7-2023.7**

目录结构：

* source - 未经处理的原始文件
  * art - 原画
  * drama - 剧本
  * ui - 界面设计
* unity - 游戏源码
* release - 编译文件
