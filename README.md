Q：这是什么类型的游戏？

A：角色扮演，解谜，恋爱（也含友情培养）

Q：游戏剧本的核心元素是什么？

A：映射现实，主要讨论的是消极情感，但整作剧本情节并非全是压抑消极的。

Q：游戏何时可以上架？

A：暂不明。

Q：游戏体验很糟糕，这是游戏最终效果吗？

A：我们会持续优化体验，但开发早期的体验确实会比较糟糕。

Q：群友的网名会出现在游戏致谢名单吗？

A：会的。
